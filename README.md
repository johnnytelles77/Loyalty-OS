# Loyalty OS – Customer Loyalty Backend Platform

Backend system for managing customers, rewards, and transactions for restaurants.

## Tech Stack
- Java 21
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA / Hibernate
- MySQL
- Swagger / OpenAPI
- JUnit, Mockito

## Architecture
Layered architecture:
- Controller – HTTP layer
- Service – Business logic
- Repository – Data access
- DTOs – Clean input/output

## Features
- Secure authentication & authorization
- Role-based access control
- Transactional reward logic
- RESTful APIs
